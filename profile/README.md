# a1neta 🗳️

Building modern campaign management infrastructure for political organizations.

## What we build
- **TMS Core** — Multi-tenant campaign management platform
- **TMS App** — Flutter mobile app for field operations
- **Platform Infra** — AWS serverless infrastructure (Lambda + API Gateway)
- **TMS DB** — Database migrations and schema management
